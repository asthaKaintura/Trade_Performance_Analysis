# Trader Performance vs Market Sentiment Analysis

## Overview

This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader performance on Hyperliquid. The aim is to understand how different market conditions affect trader behavior and profitability.

This project was completed as part of the Round-0 Data Science Internship assignment for Primetrade.ai.

---

## Datasets

- Bitcoin Fear & Greed Index
- Hyperliquid Historical Trader Data

---

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## Work Done

- Loaded and cleaned both datasets
- Removed duplicate records
- Converted timestamps into date format
- Merged the datasets using the date column
- Analyzed trader performance across different market sentiments
- Compared trade frequency, trade size, and BUY/SELL behavior
- Identified trader segments
- Created visualizations to support the findings

---

## Key Findings

- Extreme Greed showed the highest average PnL and win rate.
- Fear had the highest trading activity and total PnL.
- Average trade size was highest during Fear.
- BUY and SELL trades remained fairly balanced across different market sentiments.
